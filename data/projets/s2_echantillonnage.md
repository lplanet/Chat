# Projet Échantillonnage

**Semestre :** 2  
**Date :** Mars 2024  
**Équipe :** Binôme avec Romain Vivier  
**Technologies :** R

## Objectif du projet

Estimer la population de la région Île-de-France à partir de deux méthodes d'échantillonnage : un échantillon aléatoire simple, puis un échantillon aléatoire stratifié.

## Analyse statistique

Dans un second temps, à partir d'une autre base de données nommée 'voiture', nous avons appliqué un test d'indépendance du khi-deux, afin d'évaluer s'il existait une relation significative entre deux variables.

## Technologies utilisées

L'ensemble de l'analyse a été réalisé avec le logiciel R.
