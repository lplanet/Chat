# Projet Développement d'un Composant d'une Solution Décisionnelle

**Semestre :** 4  
**Date :** Février 2025  
**Technologies :** NodeJS, API REST, MySQL, JSON

## Contexte du projet

Amélioration de l'accès aux informations médicales issues de la Classification Commune des Actes Médicaux (CCAM), utilisée par les professionnels de santé en France.

## Objectif principal

Simplifier l'exploitation de cette base de données complexe en développant une API REST avec NodeJS.

## Fonctionnalités de l'API

Cette API permet une consultation simple, rapide et à grande échelle des données CCAM via des endpoints bien définis. Les résultats sont retournés au format JSON, assurant une compatibilité optimale avec différents systèmes et applications.

## Architecture technique

Pour le stockage des données, nous avons opté pour une base MySQL, garantissant fiabilité et performance.

## Impact du projet

Ce projet vise à offrir aux professionnels de santé un outil moderne pour accéder efficacement aux informations médicales dont ils ont besoin dans leur pratique quotidienne.
